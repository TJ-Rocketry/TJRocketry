## Description

This is a platform for the TJ Rocketry club, equipped with postgressql for storage of both user data and inventory.

For those that cannot access much of the platform due to the requirement of being a TJ student, I attached below many screenshots

<img width="2940" height="1620" alt="image" src="https://github.com/user-attachments/assets/841f1dd7-1c1a-4982-a787-de38645a4e5f" />
Students can take attendance with a code-based option, and launches can be set. This propogates the student data into a csv that can be used to easily take attendance automatically. 

<img width="2940" height="1620" alt="image" src="https://github.com/user-attachments/assets/c83ca20a-32ef-48b6-b2e8-092e38a2d34e" />
There is a full inventory system, where people can request items and get it approved by officers, and then everything is tracked.
